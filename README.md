この料金表に基づいて映画料金を決定するドメインモデルを作る。

# 設計の概要
ユーザーの属性から料金区分を決定

予約時間から曜日の区分(平日、土日祝、映画の日)と時間帯の区分(通常・レイトショー)を決める

料金区分、曜日区分、時間帯区分から最適な料金を決める
